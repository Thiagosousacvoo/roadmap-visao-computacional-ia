## 🇧🇷 Português

### 📖 Sobre o Projeto
Este repositório contém um **Roadmap Completo** e materiais de estudo detalhados para a jornada de aprendizado em **Visão Computacional e Inteligência Artificial**. O objetivo é documentar de forma estruturada desde os fundamentos da programação, passando por matemática aplicada, Deep Learning, até arquiteturas avançadas como Transformers, Segmentação de Imagens e IA Generativa.

### 🗺️ A Jornada (Fases de Estudo)
O conteúdo está organizado em **13 fases** sequenciais, cobrindo todo o espectro necessário para se tornar um Especialista:
- **Fase 00 a 01:** Fundamentos de Python e Matemática essencial para IA.
- **Fase 02 a 03:** Machine Learning e Visão Computacional Clássica (OpenCV, manipulação de imagens).
- **Fase 04 a 05:** Deep Learning e Redes Neurais Convolucionais (CNNs), incluindo Transfer Learning.
- **Fase 06 a 08:** NLP e Transformers, seguidos pelas aplicações massivas de Detecção de Objetos e Segmentação.
- **Fase 09 a 12:** Implantação de Modelos (MLOps), IA Generativa (2D e Visão 3D/NeRFs) e construção de Portfólio.

Dentro da pasta `Roadmap`, você encontrará todos os arquivos e fluxos detalhados que formam a espinha dorsal de planejamento desse aprendizado. É lá que o projeto global foi desenhado antes mesmo dos códigos existirem!

Toda a parte prática é conduzida através de **Jupyter Notebooks**, configurados com explicações visuais e ricas em markdown.

## 🇺🇸 English

### 📖 About the Project
This repository contains a **Comprehensive Roadmap** and detailed study materials for the learning journey in **Computer Vision and Artificial Intelligence**. The goal is to document in a structured way everything from programming fundamentals, applied mathematics, Deep Learning, up to advanced architectures such as Transformers, Image Segmentation, and Generative AI.

### 🗺️ The Journey (Study Phases)
The content is organized into **13 sequential phases**, covering the entire spectrum needed to become an Expert:
- **Phases 00 to 01:** Python Fundamentals and Essential Math for AI.
- **Phases 02 to 03:** Machine Learning and Classical Computer Vision (OpenCV, image manipulation).
- **Phases 04 to 05:** Deep Learning and Convolutional Neural Networks (CNNs), including Transfer Learning.
- **Phases 06 to 08:** NLP and Transformers, followed by massive applications like Object Detection and Segmentation.
- **Phases 09 to 12:** Model Deployment (MLOps), Generative AI (2D and 3D Vision/NeRFs) and Portfolio building.

### 📁 The `Roadmap` Folder
Inside the `Roadmap` folder, you will find all the detailed files and flows that form the backbone of this learning plan. This is where the overall project was designed before the code even existed!

### 📊 Progress Dashboard
To maintain motivation and track progress through the phases and modules, we built an **Interactive Dashboard** that crosses completed goals with pending objectives.
> *Open the `dashboard.html` file in your browser to see the real-time status of this Roadmap's achievements.*

### 📓 Notebooks Configuration (`.ipynb`)
All practical work is conducted through **Jupyter Notebooks**, configured with visual and markdown-rich explanations.
- They are logically divided within each Phase.
- The notebook outputs and charts have been intentionally kept to make reading via GitHub easier, without needing to run the environments firsthand.
- Be sure to install the necessary dependencies for each module (such as PyTorch, OpenCV, Transformers) if you wish to run and play with the code on your local machine.
