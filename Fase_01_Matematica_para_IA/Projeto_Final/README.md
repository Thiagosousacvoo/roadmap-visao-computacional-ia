# 🚀 Projeto Final - PCA do Zero

Sua missão é provar que você domina a Álgebra Linear aplicando o **Principal Component Analysis** sem usar Scikit-Learn.

## Arquitetura Esperada
- `src/pca.py`: Uma classe `PCA` com os métodos `fit()` e `transform()`.
- `main.py`: Lê o dataset da pasta raw, instancia seu PCA, transforma os dados e os salva na pasta processed.

Você usará conceitos de:
1. Matrizes de Covariância.
2. Autovetores e Autovalores (`np.linalg.eig`).
3. Multiplicação de Matrizes (Projeção).

Boa sorte! A solução correta separará os tipos de flores perfeitamente em 2D num gráfico de dispersão!
