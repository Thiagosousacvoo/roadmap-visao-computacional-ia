# 🏆 Projetos Finais CNN e Transfer Learning

Selecione um destes desafios práticos (ambos em script):

### Projeto 1: CNN do Zero (CIFAR-10)
- Construa sua arquitetura convolucional.
- O seu `train.py` precisa implementar Data Augmentation no dataset.
- Tente bater 85% de precisão.

### Projeto 2: Fine-Tuning ResNet (Seu Próprio Dataset)
- Faça Download de algumas fotos suas (Ex: 50 fotos de 'Caneca' vs 50 fotos de 'Garrafa').
- Use Transfer Learning congelando uma ResNet18.
- Bata 95%+ em minutos!
