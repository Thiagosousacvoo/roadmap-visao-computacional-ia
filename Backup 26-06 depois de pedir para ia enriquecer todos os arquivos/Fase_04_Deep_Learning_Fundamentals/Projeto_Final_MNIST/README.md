# 🏆 Projeto Final - Classificador MNIST PyTorch

Este é o "Hello World" oficial do Deep Learning.
Você vai classificar imagens de dígitos escritos à mão (Dataset MNIST 28x28).
A diferença é que faremos isso fora do conforto do Jupyter! O código será em formato de scripts `src/` e modular.

## Requisitos
- Uma arquitetura `MLP` usando classes `nn.Module`. (Não use CNNs ainda!).
- Treinamento rodando na GPU.
- O script `train.py` deve exibir as curvas de validação.
- Salvar o modelo `.pth` treinado!
