# 🚀 Projeto Final - Pipeline CSV Completo

> [!NOTE]
> Este projeto valida todo o conhecimento adquirido na **Fase 00**.

## Arquitetura
O projeto simula um sistema de processamento de dados para IA.
- `src/dataset.py`: Manipula I/O (Leitura de CSV) usando Pandas.
- `src/processor.py`: Aplica limpezas e filtros usando Pandas/Numpy.
- `src/analysis.py`: Responsável por gerar agregações e estatísticas.
- `src/utils.py`: Funções utilitárias como Loggers e Timing Decorators.
- `main.py`: O orquestrador central que junta todas as classes de `src/`.

**Ação Esperada**: Vá até a pasta `src/` e preencha as classes com tudo que você aprendeu!
