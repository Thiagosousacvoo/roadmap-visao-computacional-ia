# 🐙 Guia Git Essencial

> [!TIP]
> Comitar frequentemente é como "salvar o jogo".

## Comandos do Dia a Dia

1. **`git status`**: Veja quais arquivos foram modificados.
2. **`git add .`**: Adiciona TUDO que mudou na "área de stage" (pronto para comitar).
3. **`git commit -m "sua mensagem descritiva"`**: Salva o snapshot na linha do tempo local.
4. **`git push origin main`**: Envia seus commits locais para o GitHub/GitLab.
5. **`git pull`**: Puxa atualizações que outras pessoas enviaram pro servidor.

## Boas Práticas (Mensagens de Commit)
Siga o padrão *Conventional Commits*:
- `feat: Adiciona classe DatasetProcessor`
- `fix: Corrige indexOutOfBounds no slicing do numpy`
- `docs: Atualiza o README com instruções de instalação`
