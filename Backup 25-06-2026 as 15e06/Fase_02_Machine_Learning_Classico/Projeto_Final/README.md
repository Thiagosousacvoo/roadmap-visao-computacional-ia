# 🏆 Projeto Final - Pipeline Profissional

Adeus Jupyter Notebooks! Chegou a hora de estruturar Machine Learning como os Engenheiros de Dados fazem.
Seu objetivo é processar dados brutos, treinar um Pipeline do sklearn com GridSearch, e exportar as métricas em uma arquitetura de scripts limpa.

## Regras
- Execute `main.py` para disparar toda a esteira.
- `preprocessing.py` deve conter uma classe ou função que retorna o Pipeline pronto (com StandardScaler).
- O dataset simulado `housing_mock.csv` representa preços de casas baseados em tamanho, idade e criminalidade local.
