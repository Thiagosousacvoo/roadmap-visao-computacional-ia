# 🏆 Projetos Finais: Dominando a Terceira Dimensão

Sua máquina vai suar.

## Projeto 1: O Scanner 3D Caseiro
Na pasta `Projeto_01_Scanner_3D_Caseiro`, criei o esqueleto para você filmar a sua caneca na mesa, rodar o `ffmpeg` para separar as fotos, e extrair a nuvem de pontos usando um pipeline em Python chamando o COLMAP e o Open3D.

## Projeto 2: O Visualizador de Splatting
Na pasta `Projeto_02_Renderizacao_Gaussian_Splatting`, temos um framework para treinar as famosas Bolinhas Gaussianas na foto que você tirou no Projeto 1. Use ferramentas como o `nerfstudio` ou o `gsplat` para ver a mágica acontecendo e abrir o visualizador Web nativo.
